## 下載

一般使用者請下載：

- `TOOL_ASSET_NAME_Windows.zip`

通常不需要下載：

- `Source code (zip)`
- `Source code (tar.gz)`

## 支援系統

- Windows 10
- Windows 11

## 使用方式

1. 下載 ZIP 檔。
2. 將 ZIP 解壓縮到固定資料夾。
3. 雙擊資料夾中的 EXE 檔。
4. 如果 Windows 顯示安全提醒，請先確認下載來源與版本，再決定是否允許執行。

## 注意事項

- 不建議只把 EXE 檔移到桌面，除非該工具本來就是單檔版。
- 有些工具需要同一資料夾內的其他檔案。
- 如果防毒軟體隔離檔案，請先確認下載來源與版本。
- 回報問題時，請附上工具版本、錯誤截圖，以及操作步驟。

## 可替換欄位

建立 Release 時，請替換以下內容：

```text
TOOL_NAME：工具名稱，例如 PDF Toolkit
TOOL_VERSION：版本號，例如 v2.1.1
TOOL_ASSET_NAME_Windows.zip：實際下載檔名，例如 PDFToolkit_v2.1.1_Windows.zip
```
