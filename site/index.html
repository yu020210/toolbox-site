FAQ

常見問題

我需要安裝這些工具嗎？

多數工具是 portable 版本。下載 ZIP 檔後解壓縮，再執行資料夾內的 EXE 檔即可。

為什麼 Windows 會顯示安全提醒？

小型個人工具通常沒有商業程式碼簽章憑證，所以 Windows 可能會要求你確認後再執行。

GitHub 上的 Source code 是什麼？

Source code 是給開發者看的原始碼。一般使用者請下載標示 Windows、Portable、ZIP 或工具名稱的發佈檔。

可以只把 EXE 複製到桌面嗎？

不建議。有些工具需要同一個資料夾內的其他檔案。建議在桌面建立捷徑。

防毒軟體移除工具時該怎麼辦？

請先確認檔案是從本頁官方連結下載，並確認版本正確。只有在你信任來源時，才從隔離區還原。

回報問題

遇到問題時

回報問題時，請附上工具名稱、版本號、錯誤畫面截圖，以及你操作到哪一步發生問題。這樣比較容易快速判斷原因。
